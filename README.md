# URL Shortener
## Java Assignment
URL Shortener will allow users to pass a long URL and will be given a shortened URL.
That shortened URL will redirect to the complete URL.

## Tech
URL-Shortener will be built using the following technologies:
- Java
- Spring
- Maven
- Docker
